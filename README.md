# Excel: domine o editor de planilhas (Alura)

Este repositório contém meus estudos, anotações e práticas realizadas durante o curso **Excel: domine o editor de planilhas**, da plataforma Alura.

##  Sobre o curso

O curso tem como objetivo ensinar os fundamentos do Microsoft Excel, abordando desde conceitos básicos até a criação de gráficos e organização de dados.

Durante o curso, foram trabalhados:

* Estrutura de planilhas (linhas, colunas e células)
* Organização e formatação de dados
* Fórmulas e cálculos básicos
* Uso de referências
* Criação de gráficos
* Estruturação de tabelas
* Compartilhamento de planilhas

##  O que eu aprendi

* Criar e organizar planilhas do zero
* Aplicar fórmulas simples para cálculos
* Trabalhar com porcentagens e totais
* Melhorar a visualização dos dados com gráficos
* Estruturar dados de forma mais profissional

##  Estrutura do repositório

* `planilhas/` → arquivos Excel criados durante o curso
* `README.md` → descrição do projeto

##  Exemplos de projetos

Este repositório foi criado com o objetivo de consolidar meu aprendizado em Excel e servir como material de consulta futura.
